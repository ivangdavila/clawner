All notable changes to this project will be documented in this file.

### Fixed
- Rename CLI from `clawner-agent` to `clawner`
- Update all documentation references

## [0.1.1] - 2026-03-09

### Fixed
- Fix CLI binary path in npm package

## [0.1.0] - 2026-03-09

### Added
- Initial release
- WebSocket server for host connections
- REST API for host management
- Next.js dashboard for monitoring
- CLI agent (`clawner`) for connecting hosts
- Invite code authentication system
- Heartbeat telemetry (30-second intervals)
- Remote command execution (16 commands)
